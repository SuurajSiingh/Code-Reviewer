# 🔍 AI Real-time Code Reviewer

A smart web app that analyzes your code in real-time and gives AI-powered feedback to help you write better, cleaner code.

## Features
- ⚡ Instant code analysis
- 🐛 Detects bugs and errors
- 💡 Smart improvement suggestions
- 🌐 Supports multiple languages
- 🎨 Clean and simple UI

## Tech Stack
- Frontend: React.js
- Backend: Node.js
- AI: OpenAI API

## How to Run
1. Clone this repository
2. Install required packages
3. Create `.env` file and add your API key
4. Start the backend server
5. Open frontend in browser
